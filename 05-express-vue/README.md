# MEVN

## Desenvolvimento

- Cliente

  `cd ./client/`

  `npm run serve`

- Servidor

  `cd ./server/`

  `npm run dev`

- Servidor rodando no [link](http://localhost:8080)

---

## Produção

- Cliente

  `cd ./client/`

  `npm run build`

- Servidor

  `cd ./server/`

  `npm run dev`

- Servidor rodando no [link](http://localhost:8000)
